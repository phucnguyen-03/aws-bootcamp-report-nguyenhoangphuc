### Weekly Objectives:

* Learn about Amazon EC2 and its core components.
* Become familiar with AWS Identity and Access Management (IAM) for user and permission management.
* Practice launching EC2 instances and storing data using Amazon S3.
* Understand the fundamental concepts of security and resource management on AWS.

| --- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------ | --------------- | ----------------------------------------- |

### Results Achieved in Week 2:

* Acquired a solid understanding of Amazon EC2 and its primary features.

* Gained practical knowledge of the key EC2 components, including:
  * Instance Types
  * Amazon Machine Images (AMI)
  * Key Pairs
  * Amazon Elastic Block Store (EBS)

* Successfully launched an EC2 instance and established remote access using EC2 Instance Connect and SSH.

* Understood the AWS Identity and Access Management (IAM) model for managing users, groups, and permissions.

* Successfully practiced creating:
  * IAM Users
  * IAM Groups
  * IAM Policies

* Created and managed data storage using Amazon S3 buckets.

* Learned how to configure Security Groups to control inbound and outbound traffic for EC2 instances.

* Successfully completed the fundamental hands-on labs for Amazon EC2, IAM, and Amazon S3, while improving practical experience with the AWS Management Console.
