Trong suốt **12 tuần thực tập** tại **Công ty TNHH Amazon Web Services Viet Nam**, tôi đã tham gia chương trình **First Cloud AI Journey (FCAJ)** với lộ trình được chia thành hai giai đoạn chính. Giai đoạn đầu tập trung học tập và thực hành các dịch vụ cốt lõi của AWS Cloud thông qua các bài Lab và tài liệu hướng dẫn. Giai đoạn tiếp theo là phát triển dự án thực tế **AI-Powered Smart Document Assistant** trên nền tảng AWS Serverless, áp dụng những kiến thức đã học vào việc thiết kế, xây dựng và triển khai hệ thống.

Thông qua quá trình thực tập, tôi đã có cơ hội tiếp cận nhiều dịch vụ AWS như **Amazon EC2, Amazon S3, Amazon VPC, AWS Lambda, Amazon DynamoDB, Amazon Cognito, AWS AppSync, Amazon Textract, Amazon Bedrock** và nhiều công nghệ khác. Đồng thời, tôi cũng được tham gia các buổi chia sẻ kiến thức, workshop và các hoạt động thực hành nhằm nâng cao kỹ năng chuyên môn và kỹ năng làm việc trong môi trường doanh nghiệp.

Nội dung công việc được thực hiện theo từng tuần như sau:

**Tuần 1:** [Làm quen với AWS Cloud, tạo tài khoản AWS Free Tier, tìm hiểu AWS Management Console và AWS CLI.](1.1-week1/)

**Tuần 2:** [Thực hành Amazon EC2, IAM, Amazon S3 và các thao tác quản lý tài nguyên cơ bản trên AWS.](1.2-week2/)

**Tuần 3:** [Tìm hiểu Amazon VPC, Subnet, Internet Gateway, NAT Gateway, Security Group và Network ACL.](1.3-week3/)

**Tuần 4:** [Thực hành các dịch vụ lưu trữ trên AWS như Amazon S3, Storage Classes, Static Website Hosting, CloudFront và AWS Backup.](1.4-week4/)

**Tuần 5:** [Nghiên cứu các dịch vụ bảo mật AWS, IAM nâng cao, KMS, CloudTrail và Security Hub.](1.5-week5/)

**Tuần 6:** [Ôn tập kiến thức AWS, thực hành các bài Lab, thiết kế kiến trúc hệ thống Cloud và chuẩn bị kế hoạch phát triển Project.](1.6-week6/)

**Tuần 7:** [Khởi tạo dự án AI-Powered Smart Document Assistant, thiết lập AWS Amplify Gen 2, CI/CD và xây dựng kiến trúc hệ thống Serverless.](1.7-week7/)

**Tuần 8:** [Xây dựng phiên bản MVP với Amazon Cognito, Amazon S3, AWS AppSync, Amazon DynamoDB và giao diện Angular.](1.8-week8/)

**Tuần 9:** [Phát triển AI Processing Pipeline bằng AWS Lambda, Amazon Textract, Amazon Bedrock và lưu trữ Metadata trên Amazon DynamoDB.](1.9-week9/)

**Tuần 10:** [Hoàn thiện AI Processing Pipeline, triển khai AppSync Subscription, tối ưu Lambda và kiểm thử hệ thống.](1.10-week10/)

**Tuần 11:** [Phát triển Semantic Search, AI Chat theo mô hình RAG sử dụng Amazon Titan Embeddings và Amazon Bedrock Claude Haiku.](1.11-week11/)

**Tuần 12:** [Hoàn thiện hệ thống, tối ưu hiệu năng, kiểm thử tổng thể, cập nhật tài liệu kỹ thuật và chuẩn bị Demo cuối kỳ.](1.12-week12/)
