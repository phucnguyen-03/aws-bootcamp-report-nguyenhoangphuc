**On this page**, I present the worklog of my internship throughout the **12-week First Cloud AI Journey (FCAJ)** program at **Amazon Web Services Vietnam Co., Ltd.**

The internship was divided into **two main phases**:

- **Weeks 1–6:** Learning AWS Cloud fundamentals through hands-on labs, technical documentation, and guided training sessions.
- **Weeks 7–12:** Developing the **AI-Powered Smart Document Assistant** project using AWS Serverless technologies while applying the knowledge acquired during the training phase.

Throughout the internship, I gained practical experience with various AWS services, including **Amazon EC2, Amazon S3, Amazon VPC, AWS Lambda, Amazon DynamoDB, Amazon Cognito, AWS AppSync, Amazon Textract, Amazon Bedrock, AWS Amplify Gen 2, CloudWatch**, and other cloud-native technologies. I also participated in AWS workshops, technical sharing sessions, and project discussions with mentors to improve both my technical knowledge and professional skills.

The detailed work completed during each week is summarized below:

**Week 1:** [Introduction to AWS Cloud, AWS Free Tier account setup, AWS Management Console, and AWS CLI.](1.1-week1/)

**Week 2:** [Hands-on practice with Amazon EC2, IAM, Amazon S3, and basic AWS resource management.](1.2-week2/)

**Week 3:** [Learning Amazon VPC, Subnets, Internet Gateway, NAT Gateway, Security Groups, and Network ACLs.](1.3-week3/)

**Week 4:** [Exploring AWS storage services including Amazon S3, Storage Classes, Static Website Hosting, CloudFront, and AWS Backup.](1.4-week4/)

**Week 5:** [Studying AWS security services such as IAM, AWS KMS, CloudTrail, and Security Hub.](1.5-week5/)

**Week 6:** [Reviewing AWS services, practicing hands-on labs, designing cloud architecture, and preparing for the project development phase.](1.6-week6/)

**Week 7:** [Initializing the AI-Powered Smart Document Assistant project, setting up AWS Amplify Gen 2, CI/CD pipeline, and designing the serverless architecture.](1.7-week7/)

**Week 8:** [Developing the MVP by integrating Amazon Cognito, Amazon S3, AWS AppSync, Amazon DynamoDB, and the Angular frontend.](1.8-week8/)

**Week 9:** [Building the AI processing pipeline using AWS Lambda, Amazon Textract, Amazon Bedrock, and storing metadata in Amazon DynamoDB.](1.9-week9/)

**Week 10:** [Enhancing the AI processing pipeline, implementing AppSync Subscriptions, optimizing AWS Lambda, and performing integration testing.](1.10-week10/)

**Week 11:** [Developing Semantic Search and AI Chat based on Retrieval-Augmented Generation (RAG) using Amazon Titan Embeddings and Amazon Bedrock Claude Haiku.](1.11-week11/)

**Week 12:** [Optimizing system performance, conducting final testing, preparing technical documentation, and getting ready for the final project demonstration.](1.12-week12/)
